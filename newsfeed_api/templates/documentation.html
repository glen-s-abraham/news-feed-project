News feed API

Functionality

This is a simple news feed API. The functionality includes retrieval of news feed
in all categories,selected ones,or a particular article in detail.

Author

Glen S Abraham

glenprojects666@gmail.com

Usage

/api/news_feed
Fetches all the available news in the order of latest first
/api/news_feed/id
Returns the descreptive view of a particular news article
/api/news_feed/?cat=#category#
Fetches specific news belonging to a category in the order of latest first
Categories
BR-'Breaking'
IN-'International'
NT-'National'
ST-'State'
DT-'District'
LC-'Local'
CL-'Classified'
TC-'Technology'
SP-'Sports'.
(The parameters are case insensitive)
Ex /api/news_feed/?cat=nt
